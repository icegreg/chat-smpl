    Call:
      type: object
      properties:
        type:
          type: string
          enum: [p2p, meeting]
          description: Тип звонка (P2P или мероприятие)
          example: "p2p"
        status:
          type: string
          enum: [ringing, active, completed, declined, missed, cancelled, failed] 
          description: |
            Статус звонка:
            - ringing: звонок идёт, ожидается ответ
            - active: звонок принят и идёт
            - completed: звонок завершён успешно
            - declined: звонок отклонён получателем
            - missed: звонок не был принят (таймаут)
            - cancelled: звонок отменён инициатором до ответа
            - failed: технический сбой при установлении связи
          example: "completed"
        direction:
          type: string
          enum: [inbound, outbound]
          nullable: true
          description: |
            Направление звонка относительно текущего пользователя (только для P2P):
            - inbound: входящий звонок
            - outbound: исходящий звонок
            - null: для ВКС-мероприятий не применимо
          example: "outbound"
        from:
          type: string
          format: uri
          nullable: true
          description: |
            URI инициатора звонка. Поддерживаемые схемы:
            - user: внутренний пользователь (user:123@domain.com) 
            - tel: телефонный номер (tel:+79991234567)
            - sip: SIP-адрес (sip:user@domain.com)
            null для мероприятия
          example: "user:123@domain.com"
        to:
          type: string
          format: uri
          nullable: true
          description: |
            URI получателя звонка. Поддерживаемые схемы:
            - user: внутренний пользователь (user:456@domain.com) 
            - tel: телефонный номер (tel:+79991234567)
            - sip: SIP-адрес (sip:user@domain.com)
            null для мероприятия
          example: "user:456@domain.com"
        displayName:
          type: string
          description: |
            Отображаемое имя собеседника:
            - для P2P: имя того, кто на другом конце (кому звоню или кто звонит)
            - для мероприятия: название мероприятия
          example: "Пётр Петров"
        createdAt:
          type: string
          format: date-time
          description: Время создания/инициации звонка
          example: "2025-10-31T10:00:00Z"
        answeredAt: 
          type: string
          format: date-time
          nullable: true
          description: Время ответа на звонок (null если не был принят)
          example: "2025-10-31T10:00:15Z"
        endedAt: 
          type: string
          format: date-time
          nullable: true
          description: Время окончания звонка (null для активных)
          example: "2025-10-31T10:30:00Z"
        duration: 
          type: integer
          nullable: true
          description: Длительность разговора в секундах (null для не принятых или активных)
          example: 1785
        declineReason:
          type: string
          nullable: true
          enum: [user_declined, busy, unavailable, timeout, technical_error]
          description: |
            Причина отклонения/пропуска звонка:
            - user_declined: пользователь отклонил звонок
            - busy: пользователь занят другим звонком
            - unavailable: пользователь недоступен (оффлайн)
            - timeout: превышено время ожидания ответа
            - technical_error: техническая ошибка
          example: "user_declined"
        callMediaType: 
          type: string
          enum: [audio, video]
          nullable: true
          description: Тип медиа в звонке
          example: "video"
      required:
        - type
        - status
        - answeredAt
        - direction
        - displayName
        - duration
        - endedAt
